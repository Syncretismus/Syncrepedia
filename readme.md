## 緣起
本項目是 [Syncretismus](https://github.com/Syncretismus/Syncretismus) 的延伸，以百科工具書爲主要收錄對象。這些辭書中，少數爲全文 OCR 結果，多數爲「詞條—頁碼—原書圖像」模式，極少數爲前二者之結合。
